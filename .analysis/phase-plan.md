# Scope

Only two phases are in scope:

1. Provider registry, adapters, diagnostics, model discovery, URL normalization, SSRF controls, API routes, UI and tests.
2. PostgreSQL/Supabase with Drizzle ORM, migrations, typed repositories, auth/session/provider/chat/message/agent/workspace persistence and tests.

Agent streaming expansion, unrelated UI features, and later product phases are explicitly out of scope.
